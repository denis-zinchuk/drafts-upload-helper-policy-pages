Privacy Policy

Last updated: June 17, 2026

Drafts Upload Helper is a personal desktop tool used to upload user-selected video files to TikTok through the official TikTok Content Posting API.

The tool may request TikTok authorization from the user. The authorization token is used only to upload content selected by the user to the user's TikTok account.

The tool does not sell personal data. The tool does not share personal data with third parties except TikTok, as required for the upload process.

The tool may process video files selected by the user for upload. These files are sent to TikTok only when the user runs the upload process.

Access tokens and refresh tokens should be stored locally and securely by the user. They should not be shared publicly or committed to public repositories.
